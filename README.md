# PFC Calendar

## 概要
筋トレやダイエットなどの体作りをしている人が, 簡易的に食事管理をするためのWEBアプリケーション.

<dl>
  <dt>Protein</dt>
  <dd>タンパク質</dd>
  <dt>Fat</dt>
  <dd>脂質</dd>
  <dt>Carbohydrate</dt>
  <dd>炭水化物</dd>
</dl>

食事に含まれるPFCの量やバランスは異なり, また体作りの目的毎に必要なPFCも考える必要がある.
そのような計算を簡易的に行えるようにツール化し, カレンダーとして日毎に管理するのが目的.
